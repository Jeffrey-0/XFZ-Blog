**浏览器内核**

主要分为四种：**Trident、Gecko、Blink、Webkit**

| 浏览器名称 | 内核                                      |
| ---------- | ----------------------------------------- |
| IE         | Trident内核，也称IE内核                   |
| Chrome     | Webkit内核，现在是Blink内核               |
| Firefox    | Gecko内核，俗称Filefox内核                |
| Safar      | Webkit内核                                |
| Opera      | 最初Presto内核，加入谷歌后，Webkit->Blink |
| 360、猎豹  | IE+Chrome双内核                           |
| 百度       | IE内核                                    |
| QQ浏览器   | Trident(兼容模式) + Webkit（高速模式）    |
