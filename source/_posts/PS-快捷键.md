## 编辑文件快捷键

| 操作     | 快捷键       |
| -------- | ------------ |
| 撤销     | CTRL+Z       |
| 向前一步 | CTRL+SHIFT+Z |
| 向后一步 | CTRL+ALT+Z   |
| 退取     | CTRL+SHIFT+F |
| 剪切     | CTRL+X       |
| 复制     | CTRL+C       |
| 合并复制 | CTRL+SHIFT+C |
| 粘贴     | CTRL+V       |
| 原位粘贴 | CTRL+SHIFT+V |
| 自由变换 | CTRL+T       |
| 再次变换 | CTRL+SHIFT+T |
| 色彩设置 | CTRL+SHIFT+K |
